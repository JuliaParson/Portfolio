Vitalcare Pharmacy Registration Form

About the Project

Vitalcare Pharmacy is a mock pharmacy I created for a personal project. I wanted to explore the creation of forms in HTML so I decided to make a registration form using HTML, CSS, and PHP. While filling out the form you are prompted for any mandatory fields and if any are missed it will not allow you to submit the form. This project challenged my problem-solving skills and highlighted my passion for learning new things.
